# Bài tập Trí tuệ Nhân tạo

Dự án này bao gồm các bài toán AI cơ bản:

1. **Trí tuệ nhân tạo trong trò chơi**
2. **8-Puzzle với A* (Manhattan Heuristic)**
3. **Bài toán Người du lịch (TSP) với Simulated Annealing**
4. **Giải mê cung với BFS, DFS và A***
5. **Hệ chuyên gia chẩn đoán bệnh**
6. **Hệ suy luận cây gia phả (First-Order Logic)**
7. **Phát hiện thư rác với Naive Bayes**

## Cấu trúc thư mục

```
├── 1_game_ai/          # Trí tuệ nhân tạo trong trò chơi
├── 2_8puzzle/          # Giải 8-Puzzle bằng A*
├── 3_tsp/              # Bài toán TSP với Simulated Annealing
├── 4_maze/             # Giải mê cung với BFS, DFS, A*
├── 5_expert_system/    # Hệ chuyên gia chẩn đoán bệnh
├── 6_family_tree/      # Hệ suy luận cây gia phả
└── 7_spam_detection/   # Phát hiện thư rác với Naive Bayes
```
