# Trí tuệ nhân tạo trong trò chơi - Tic Tac Toe

## Mô tả
Game Tic Tac Toe với AI sử dụng thuật toán Minimax và Alpha-Beta Pruning.

## Thuật toán
- **Minimax**: Tìm nước đi tối ưu bằng cách duyệt cây game
- **Alpha-Beta Pruning**: Tối ưu hóa bằng cắt tỉa các nhánh không cần thiết

## Chạy chương trình
```bash
python tic_tac_toe.py
```

Bạn sẽ chơi với AI không thể thắng được!
