# Giải mê cung với BFS, DFS và A*

## Mô tả
So sánh 3 thuật toán tìm đường trong mê cung.

## Thuật toán
- **BFS**: Tìm kiếm theo chiều rộng - đảm bảo đường đi ngắn nhất
- **DFS**: Tìm kiếm theo chiều sâu - nhanh nhưng không tối ưu
- **A***: Tìm kiếm có heuristic - tối ưu và hiệu quả

## Chạy chương trình
```bash
python maze_solver.py
```
